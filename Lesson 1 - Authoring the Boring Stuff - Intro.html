In a nutshell this chapter is just an introduction to Python. Author explains that this book is not designed to teach you how to program. It is designed to teach people who want to get up to speed coding small programs that do practical tasks as soon as possible.¶

Author explains that you will need to download and install Python located at https://www.python.org/ there are two versions the legacy 2 and the new Python 3. Python 3 is highly recommended by author and in most blogs and articles.¶

Author explains that IDLE editor application and the IDLE text editor are installed with the download.¶
